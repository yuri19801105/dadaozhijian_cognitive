# ADR-0001: 采用"太极→十方"数字宇宙观作为规范架构映射

## 状态

已接受

## 背景

项目需要一个统一、自洽的认知架构来组织 AI 系统的各功能模块。候选方案有:(a) 直接套用现代模块名(planner / memory / tool-use);(b) 采用"太极→十方 + 河图洛书/罗盘/算盘"的数字宇宙观作为映射框架。

## 决策

采用方案 (b)。将太极至十方(共 10 个演化层级)及三个固件层(河图洛书、罗盘、算盘)逐一映射为 AI 认知系统的功能模块,形成"从元认知到具体执行"的完整流水线。所有模块命名与术语以 `CONTEXT.md` 为唯一真相源。

## 理由

- 该序列从抽象到具体、从静态到动态逐级演化,天然契合"元认知 → 执行 → 反馈"的闭环。
- 补全六合与七星后,架构从单纯内在认知循环扩展到含环境交互与动态演化的完整系统。
- 统一词汇避免后续实现层的命名漂移。

## 影响

- 所有 issue、模块、变量命名须引用 `CONTEXT.md` 术语。
- 新增概念须经 `/domain-modeling` 决议,不得擅自发明。
