# ADR-0003: 模块契约语言无关,技术栈实现阶段再定

## 状态

已接受

## 背景

本轮交付物为**设计文档套件**,不含运行代码。模块内部算法与实现语言尚未确定,但需先固化模块间的契约,以免后续实现各自为政。

## 决策

- 所有模块仅以 `职责 / 输入端口 / 输出端口 / 不变量 / 消息` 描述,签名用伪类型(如 `Tensor[6]`、`Sequence[Task]`),不绑定具体语言。
- 技术栈(语言、框架、运行时)推迟到实现阶段选型,届时经独立 ADR 记录。

## 理由

- 用户明确"暂不定"技术栈;过早绑定会限制后续方案空间。
- 语言无关的契约足以指导实现阶段的接口设计与测试边界。

## 影响

- 实现阶段引入语言/框架时,须另立 ADR 记录选型与理由。
- 契约变更须同步回写 `docs/architecture.md` 与 `CONTEXT.md`。
