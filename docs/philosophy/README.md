## 整体脉络：一条"分化—落地"的宇宙生成链

这十个概念并非孤立并列，而是一条**层层展开、环环相扣**的逻辑链：

```
        太极（本原·一）
          │ 一分为二
        两仪（阴阳·二）
          │ 分立层级 / 各分阴阳
   三才（天地人·三）   四象（四态·四）
          │                │ 加中土 / 各分阴阳
          │            五行（生克·五）   八卦（象数·八）
          │                │                │ 加中宫
   六合（空间·六）─────────┤            九宫（方阵·九）
          │ 天文枢机                          │
   七星（北斗·定序）─────────────────────────┤
          │ 空间细化至极                       │
        十方（周遍·十）←── 呼应 ── 太极（至大无外）
```

- **由一到多**：太极（一）→两仪（二）→…→十方（十），是数目的递增，更是**从浑然本原到具体周遍的次第分化**。
- **体用相贯**：太极为体，越往后越是"用"——越具体、越可操作（八卦可推演、九宫可运算、十方可执行）。
- **首尾圆合**：太极"至大无外"的本原，与十方"周遍无尽"的空间，一始一终、遥相呼应，构成闭环。

### 三条贯穿性主线

1. **分化主线（象数）**：太极→两仪→四象→八卦，是标准的二进制分化（1→2→4→8），八卦加中宫成九宫。
2. **关系主线（要素）**：两仪（阴阳）＋五行（生克），编织万物的对待与生克网络。
3. **空间主线（方位）**：三才（上中下）→四象（四方）→六合（六向）→十方（十向），由层级到周遍的空间细化；七星、九宫是其中的天文枢机与地面方阵。
