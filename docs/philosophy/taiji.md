# 太极（tàijí）

> 「易有太极，是生两仪，两仪生四象，四象生八卦。」——《周易·系辞上》

太极是中国哲学中层级最高、最具统摄性的宇宙本体概念，居"太极→两仪→三才→四象→五行→六合→七星→八卦→九宫→十方"这一序列的源头。它既是万物未分的浑沌本原，也是阴阳运化的总枢纽。

---

## 一、哲学含义与核心思想

**"太"者，至也、极也；"极"者，枢极、终极。** 合而言之，"太极"指宇宙生成之前那个至大无外、至微无内、阴阳未判的本原状态，是一切存在的终极根据。

其核心思想可归纳为三层：

1. **本体论上的"一"**：太极是"一"，是未分化的整全。它先于天地、先于阴阳，是"有物混成，先天地生"（《道德经》）的那个混沌本体。宋儒周敦颐《太极图说》开篇即言"无极而太极"，把它推到超越有无的终极高度。

2. **生成论上的"生生"**：太极不是静止的实体，而是内含运动之几的动态本原。"太极动而生阳，动极而静，静而生阴"——阴阳二气由太极一体两分而来，由此开启"生生不息"的宇宙创化。**太极之"生"不是外在制造，而是自身内在的分化与展开。**

3. **方法论上的"执两用中"**：太极图（阴阳鱼）以黑白双鱼相抱、彼此含藏（阳中有阴、阴中有阳）、首尾相衔的图形，浓缩了"对立统一、消长转化、中和平衡"的辩证智慧。它启示人以整体观、动态观、平衡观看待世界。

---

## 二、历史渊源与发展脉络

- **先秦奠基**："太极"一词首见于《庄子·大宗师》"在太极之先而不为高"，但作为宇宙生成范畴确立于《周易·系辞》"易有太极，是生两仪"。此时"太极"与"道""一""太一"等本原概念相互呼应。
- **汉唐诠释**：汉代以"元气"释太极（如郑玄），把太极理解为混沌未分的元气；唐代孔颖达《周易正义》承此说，谓"太极谓天地未分之前，元气混而为一"。
- **宋代哲学化高峰**：周敦颐作《太极图说》，以"无极—太极—阴阳—五行—万物"构建完整宇宙生成图式，太极由此成为理学的核心范畴。朱熹进一步以"理"释太极——"太极只是一个理字""总天地万物之理，便是太极"，把它提升为最高的形而上本体。同时期陆九渊、后世王阳明则更强调太极与心性的贯通。
- **图学与实践流布**：太极图（阴阳鱼）经宋元道教与理学传播，成为中华文化最具辨识度的符号；明清之际，"太极"之名又落实于养生武术（太极拳），把哲理转化为身体实践。

---

## 三、在东方哲学体系中的地位与作用

**太极是整个东方宇宙论的"第一因"和"总纲"。** 它在体系中承担三重作用：

- **本原**：为万物提供终极来源，回答"世界从何而来"。
- **枢纽**：作为阴阳未分而含阴阳的临界点，它是"一"通向"多"的转换枢纽——没有太极，两仪无从生，后续序列全部失去起点。
- **准则**：太极所示的"中和—平衡—转化"成为儒、道、医、术各家共享的方法论原则，是"大道至简"的最高表达。

在儒家它是"理"之极致，在道家它近于"道"与"一"，在道教内丹与医家它是"先天一气"，三教于此汇流。

## 四、与其他九个概念的关联关系

太极是序列的**根**，其余九者皆由它次第展开：

- **太极 → 两仪**：太极一分为二，生阴阳（两仪），这是最直接、最根本的派生关系——"是生两仪"。
- **太极 → 三才**：阴阳交而立天、地、人三才，太极是三才所共禀的一本。
- **太极 → 四象**：两仪各分阴阳而成四象（太阳、太阴、少阳、少阴），太极是其总源。
- **太极 → 五行**：周敦颐图式中"阳变阴合而生水火木金土"，五行是太极经阴阳而落实的五种质料动能。
- **太极 → 六合 / 七星 / 八卦 / 九宫 / 十方**：这些是太极之理在空间（六合、十方）、天象（七星）、象数（八卦、九宫）等不同层面的具体铺展。**太极为体，余者为用**——愈往后愈具体、愈可操作，而其理一以贯之。

一句话：**太极是"一"，两仪以下是"一"的层层分化与场景化落地。**

---

## 五、对后世哲学、文化与实践的影响

1. **哲学**：奠定了中国"体用一源、一多相摄"的本体思维范式；宋明理学、道教内丹学、易学象数学皆以太极为逻辑起点。
2. **符号文化**：太极图成为中华乃至东亚文化的标志性图腾，广泛见于建筑、器物、旗帜、品牌，并被现代科学（如玻尔以太极图喻互补原理）借喻。
3. **医学养生**：中医"阴阳平衡"理论、内丹"炼精化气"皆本于太极；太极拳、太极剑把"以柔克刚、动静相生"的哲理化为养生与技击之道。
4. **现代启示**：其对立统一、动态平衡、整体关联的思想，与系统论、控制论、复杂性科学乃至本项目"大道至简认知架构"的顶层设计相通——**以一个最简的本原，统摄层层展开的复杂系统。**
